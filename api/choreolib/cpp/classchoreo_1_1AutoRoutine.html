Detailed Description

template<choreo::TrajectorySample SampleType, int Year = util::kDefaultYear>
class choreo::AutoRoutine< SampleType, Year >

A loop that represents an autonomous routine.

This loop is used to handle autonomous trigger logic and schedule commands. This loop should not be shared across multiple autonomous routines.

Template Parameters
SampleTypeThe type of samples in the trajectory.
Thefield year. Defaults to the current year.
